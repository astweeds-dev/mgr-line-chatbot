# คู่มือพนักงาน — ระบบสั่งอาหาร MGR

## เข้าใช้งาน

1. เจ้าของจะส่ง **ลิงก์แดชบอร์ด** มาให้ทาง LINE (ได้รับทุกครั้งที่ระบบเปิด)
2. กดลิงก์ → จะเข้าหน้า **"เข้าสู่ระบบพนักงาน"**
3. พิมพ์ **รหัส PIN** ที่เจ้าของให้มา แล้วกด **"เข้าสู่ระบบ"**
4. เข้าได้แล้ว! ครั้งต่อไปไม่ต้องพิมพ์ใหม่ (จำไว้ในเครื่อง)

> ถ้าเจ้าของยังไม่ส่งลิงก์มา → รอก่อน อย่าใช้ลิงก์เก่า (เปลี่ยนทุกครั้งที่เปิดเครื่อง)

---

## หน้าจอหลัก

มี 3 แท็บ:
- **ออเดอร์** — ดูและจัดการออเดอร์ที่เข้ามา (ใช้แท็บนี้เป็นหลัก)
- **ประวัติ** — ดูออเดอร์ที่เสร็จแล้ว
- **เมนู** — จัดการเมนูอาหาร (เฉพาะเจ้าของ ไม่ต้องแตะ)

---

## วิธีจัดการออเดอร์ (ทำตามปุ่มเดียว!)

ทุกออเดอร์จะมี **ปุ่มใหญ่ 1 ปุ่ม** บอกว่าต้องทำอะไรต่อ ทำตามปุ่มไปเรื่อยๆ:

### ขั้นตอนที่ 1: ลูกค้าสั่ง + โอนเงิน
- การ์ดจะขึ้นว่า **"⏳ รอลูกค้าโอนเงิน / ส่งสลิป"**
- ยังไม่ต้องทำอะไร รอลูกค้าส่งสลิปมา

### ขั้นตอนที่ 2: ลูกค้าส่งสลิปแล้ว → ยืนยันรับเงิน
- การ์ดจะมี **รูปสลิป + ยอดเงินตัวใหญ่**
- ถ้ามีเสียง **"บี๊บ บี๊บ"** + การ์ดกระพริบ = ออเดอร์ใหม่เข้ามา!
- **ดูสลิป** ตรวจว่ายอดถูกต้อง (กดรูปสลิปดูใหญ่ได้)
- กดปุ่ม **"✅ ยืนยันรับเงิน"**
- ระบบจะแจ้งลูกค้าอัตโนมัติว่าได้รับเงินแล้ว

> หมายเหตุ: ถ้าระบบตรวจสลิปอัตโนมัติผ่าน (SlipOK) จะข้ามขั้นนี้ไปเลย ไม่ต้องกดยืนยัน

### ขั้นตอนที่ 3: เริ่มทำอาหาร
- กดปุ่ม **"🍳 เริ่มทำอาหาร"**
- ถ้าอยากบอกลูกค้าว่ารอกี่นาที → กดเลือก **10 / 20 / 30 นาที** ก่อนกดปุ่ม (ไม่บังคับ)
- ระบบจะแจ้งลูกค้าอัตโนมัติ

### ขั้นตอนที่ 4: ออกไปส่ง
- กดปุ่ม **"🛵 ออกไปส่ง"**
- เลือกเวลาได้ (ไม่บังคับ)
- ระบบจะแจ้งลูกค้าอัตโนมัติ

### ขั้นตอนที่ 5: ส่งสำเร็จ
- กดปุ่ม **"✅ ส่งสำเร็จ"**
- ออเดอร์จะย้ายไปแท็บ **"ประวัติ"** เป็นอันจบ!

---

## สรุปง่ายๆ

```
ลูกค้าสั่ง → รอสลิป → ยืนยันรับเงิน → เริ่มทำ → ออกส่ง → ส่งสำเร็จ → จบ!
```

กดปุ่มที่ขึ้นมาไปเรื่อยๆ แค่นั้น!

## ถ้าเจอปัญหา

| ปัญหา | วิธีแก้ |
