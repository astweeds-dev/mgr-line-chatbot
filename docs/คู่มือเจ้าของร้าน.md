# คู่มือเจ้าของร้าน — ระบบสั่งอาหาร MGR

## ภาพรวมระบบ

ระบบสั่งอาหารออนไลน์ผ่าน LINE ทำงานบนคอมพิวเตอร์ที่ร้าน:
- **ลูกค้า** สั่งผ่าน LINE → ระบบเปิดหน้าสั่งอาหาร → จ่ายเงินผ่าน QR PromptPay → ส่งสลิป
- **พนักงาน** ดูออเดอร์ในแดชบอร์ด (เปิดในมือถือ/แท็บเล็ต) → กดยืนยันรับเงิน → ทำอาหาร → ส่ง → เสร็จ
- **ลูกค้า** ได้รับแจ้งเตือนสถานะทุกขั้นตอนอัตโนมัติผ่าน LINE

---

## เปิด/ปิดระบบ

### เปิดระบบ
1. เปิดคอมพิวเตอร์
2. ดับเบิลคลิก **`start-all.bat`** ในโฟลเดอร์ `mgr-line-chatbot`
3. รอจนหน้าต่างสีดำขึ้นข้อความ → ระบบทำงานเอง
4. เจ้าของจะได้ **LINE แจ้งเตือน "🟢 ระบบ MGR เปิดทำงานแล้ว"** พร้อมลิงก์แดชบอร์ดพนักงาน
5. **ส่งต่อลิงก์แดชบอร์ดให้พนักงาน** ทาง LINE (ต้องส่งทุกครั้งที่เปิดเครื่อง เพราะลิงก์เปลี่ยน)

### ปิดระบบ
- **ปิดหน้าต่างสีดำ** (กากบาท ✕) → ระบบจะส่ง LINE "🔴 ระบบ MGR ปิดตัวลง" แล้วปิดทุกอย่าง

### ถ้าระบบล่ม
- ระบบจะ **restart อัตโนมัติ** ภายใน 5 วินาที ไม่ต้องทำอะไร
- ถ้าเน็ตหลุด → เปิดเน็ตกลับมา → ระบบต่อเองอัตโนมัติ
- ถ้าคอมดับ → เปิดคอมแล้วดับเบิลคลิก `start-all.bat` ใหม่

## ให้พนักงานเข้าใช้งาน

1. หลังระบบเปิด → ได้ LINE แจ้งเตือนพร้อมลิงก์
2. ส่งต่อลิงก์ **แดชบอร์ดพนักงาน** ให้พนักงานทาง LINE
3. บอก PIN: **`1212312121`** (เปลี่ยนได้ ดูหัวข้อ "ตั้งค่า" ด้านล่าง)
4. พนักงานเปิดลิงก์ → พิมพ์ PIN → ใช้งานได้เลย

> ⚠️ ลิงก์เปลี่ยนทุกครั้งที่เปิดเครื่อง → ต้องส่งลิงก์ใหม่ให้พนักงานทุกครั้ง
> ถ้าไม่อยากให้ลิงก์เปลี่ยน → ซื้อโดเมน (~350 บาท/ปี) แล้วตั้ง Named Tunnel (แจ้งผู้ดูแลระบบ)

## จัดการเมนู

เจ้าของจัดการเมนูเองได้ที่แท็บ **"เมนู"** ในแดชบอร์ด:

### เพิ่มเมนูใหม่
1. กดปุ่ม **"+ เพิ่มเมนูใหม่"**
2. กรอกข้อมูล:
   - **ID** — เลขประจำเมนู (ห้ามซ้ำ เช่น 50, 51, 52...)
   - **หมวด** — เลือก: อาหาร / กาแฟ / นม & ชา / โซดา
   - **ชื่อ (ไทย)** / **ชื่อ (EN)** — ชื่อเมนู 2 ภาษา
   - **ราคา** — ใส่เป็น JSON เช่น:
     - ราคาเดียว: `{"default":70}`
     - เลือกเนื้อสัตว์: `{"หมู":70,"เนื้อ":80}`
     - เลือกเมล็ดกาแฟ: `{"คั่วอ่อน":70,"คั่วเข้ม":70}`
3. กด **"บันทึก"** → เมนูขึ้นหน้าลูกค้าทันที!

### แก้ไข / ลบเมนู
- กดปุ่ม **✏️** ข้างเมนูที่ต้องการแก้
- กดปุ่ม **🗑** เพื่อลบ
- กดปุ่ม **🔒 ปิด / 🔓 เปิด** เพื่อซ่อน/แสดงเมนูชั่วคราว (ไม่ต้องลบ)

> เมนูที่แก้ไข/เพิ่ม/ลบ จะมีผลกับหน้าสั่งของลูกค้า **ทันที** ไม่ต้อง restart
